=============================[Connect & Update Remote Repo]========================


>> git clone <URL> 
------------------------------------------------------------------------------------------------------------------------------------------------
create a local repo from an exisiting remote repo 
#use it : only at the first time 
-------------------------------------------------------------------------------------------------------------------------------------------------


>> git pull origin
------------------------------------------------------------------------------------------------------------------------------------------------
update your ( local repo) from the ( remote repo) 
use one of them
#use it : when there someone update something on the remote repo related to your branch
-------------------------------------------------------------------------------------------------------------------------------------------------


>> git push origin
-------------------------------------------------------------------------------------------------------------------------------------------------
update (remote repo) from the (local repo)
#use it : after adding --> commiting --> push it to the remote repo
-------------------------------------------------------------------------------------------------------------------------------------------------
